# @inedge/designer

A React canvas design library for building Figma-style editors — frames, layers, actions, unit system, print bounds. Composable API, type-safe end to end, performant at 120 Hz drag cadence.

## Status

🚧 Under active development. See the [implementation roadmap](../../.claude/plans/cozy-gliding-meerkat.md) for the phased plan.

## Non-goals

- Not for >2000 layer / 10 000+ complex-element scenes — use Framer/Canva/Figma for that class.
- Not a WebGL/WebGPU renderer.
- Not a collaborative real-time editor.
- Multi-page and layer groups are deferred to v1.1+.
